# Git

1. Check the git version

```bash
git --version
```

2. Configure Git

- Configure the username

```bash
git config --global user.name "[username]"
git config user.name
```

- Configure email

```bash
git config --global user.email "email"
git config user.email
```

3. Initialize a Repository

```bash
git init
```

4. Legends

- U : Untracked
- A : Added
- M : Modified
